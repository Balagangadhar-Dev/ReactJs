1. Node.js Installation

You'll need to have Node.js version 14.0 or higher installed on your local development machine. You can check your current version by running the following command in your terminal:

node -v

If you don't have Node.js installed, you can download it from the official website. The installer for Node.js also includes npm (Node Package Manager).

Download Node.js →

2. Creating a New React App

To get started, we'll use the official `npx create-react-app` command. This command sets up a new React project with all the necessary build configurations pre-configured, so you can start coding right away.

Open your terminal or command prompt, navigate to the directory where you want to create your app, and run the following command. This will create a directory named `my-app` inside the current folder.

npx create-react-app my-app
Read more about creating an app →

3. Project Structure

After the installation is complete, a new directory is created with a pre-defined structure. This structure is minimal but contains everything you need to build a single-page application.

  • my-app/
  • ├── README.md
  • ├── node_modules/
  • ├── package.json
  • ├── public/
  • │ ├── index.html (The template HTML file)
  • │ └── ...other static assets
  • └── src/
  • ├── App.js (Your main component)
  • ├── index.js (The JavaScript entry point)
  • └── ...other source files
Explore the folder structure →

4. Available Scripts

Inside the new project directory, you'll find a set of built-in scripts that simplify the development process. Navigate into the folder and run the commands as needed.

cd my-app

npm start

This runs the app in development mode. Open http://localhost:3000 to view it in the browser. The page will reload if you make edits.

npm run build

This command builds the app for production to the `build` folder. It bundles React in production mode and optimizes the build for the best performance.
